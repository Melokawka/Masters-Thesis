# Synthetic Dynamic Camera Shot Generator (Unity)
An open-source Unity-based synthetic video generator for modeling and parameterizing dynamic camera motion to support deep learning research in shot classification.

![ezgif-82297872ecf0af50](https://github.com/user-attachments/assets/9c667a7a-38f4-4680-87e5-3df6f288d1fd)

## Research Contributions

- Parametric model of dynamic camera movement for synthetic generation
- Unity-based configurable video clip generator
- Prototype synthetic dataset for dynamic shot classification
- Evaluation using 3D CNNs and Video Transformers
- Proposed automated synthetic data quality scoring method
